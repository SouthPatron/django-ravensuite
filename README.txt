===========
django-ravensuite
===========

django-ravensuite is a set of common utilities used by all apps.

Installation
============
- From pypi:
    $pip install django-ravensuite

- From source:
    $python setup.py install

Configuration
=============

Use it like crazy.

RUN   
===
$python manage.py runserver
